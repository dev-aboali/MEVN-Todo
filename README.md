# MEVN-Todo
Todo Full-stack app using node.js and vue.js

## Technologies
- Node.js
- Express.js
- MongoDB & Monk
- bootstrap4
- Animate.css
- Joi

### version
1.0.0

### installation
##### Client installation
$ cd client

$ npm install


##### Server installation
$ cd server

$ npm install


#### Running 
##### Running Server 
$ nodemon

##### Running Client
$ npm run serve
